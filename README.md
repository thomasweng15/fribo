# fribo

## Getting started

### Shopping List
* Raspberry Pi 3 w/power adapter
* CdS photoresistor
* LED Display (no touch)
* Ultrasonic sensor
* 2x speakers
* 2x microphones
* Sound sensor

### Setting up the Raspberry Pi

1. Install Raspbian
2. Install ROS Indigo
3. Set up catkin workspace
3. Install this repo in workspace
